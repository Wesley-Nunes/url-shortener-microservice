URL Shortener

How to Use the API

To shorten a URL, make a POST request to the following endpoint with a JSON payload:

POST /api/shorturl

Example payload:

{
	"url": "https://www.example.com"
}

Example response:

{
	"original_url": "https://www.example.com",
	"short_url": "1"
}

Accessing: /api/shorturl/1

Will Redirect to: https://www.example.com
